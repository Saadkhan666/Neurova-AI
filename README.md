# Neurova OS (Saad Khan Edition)

A high-performance, single-file AI operating system web app.

**Features:**
* **Gemini/DeepSeek Integration:** Uses OpenRouter API.
* **Holo-Canvas:** Renders HTML/SVG artifacts in real-time.
* **Cloud Sync:** Syncs chats via Firebase.
* **PWA Ready:** Installable on Android/iOS.
* **Owner Mode:** Hardcoded API keys for ease of use.
